В этой папке хранятся файлы JavaScript и CSS шаблона.


Если нужно поменять шаблон, то нужно заменить эти файлы на другие.
Шаблоны хранятся в БД и путем копирования файлов из БД меняется вид сайта.


В редакторе создания шаблона создается код JavaScript и CSS по данным параметрам.

Поля БД СОКРАЩЕННОГО шаблона:
ID шаблона
Имя шаблона
Ширина страницы
Высота страницы
Путь к CSS файлу шаблона
Путь к Javascript файлу шаблона

Ширина хедера
Высота хедера
Отключить хедер

Ширина левой колонки
Высота левой колонки (Если 100%, то колонка по всей высоте страницы)
Отключить левую колонку

Ширина правой колонки
Высота правой колонки (Если 100%, то колонка по всей высоте страницы)
Отключить правую колонку

Ширина футера
Высота футера
Отключить футер
